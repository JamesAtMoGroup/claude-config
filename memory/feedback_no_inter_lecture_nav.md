Never include "上一堂：Lecture N ／ 下一堂：Lecture N" (or any prev/next lecture navigation) in n8ncourse lecture pages.

**Why:** James explicitly does not want cross-lecture navigation labels. Raised this more than once — it keeps appearing and must not be added.

**How to apply:** When building or editing any lectureN/index.html, do not add any UI element that links to the previous or next lecture by label. Navigation back to the main site is via the "← 返回主頁" top-bar link only.
